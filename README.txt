Ce github permet d'avoir accès à :

- un dossier src comprenant tous les fichiers exécutant le jeu de l'othello 

- un dossier obj comprenant les exécutables

- un dossier installation comprenant les explications pour installer la SDL

- un dossier doc comprenant les fichiers Doxygen

- un dossier datafile comprenant les images utilisées dans le programme

- un dossier bin comprenant l'exécutable général de l'othello

- un makefile permettant d'exécuter le programme
